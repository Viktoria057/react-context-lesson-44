# 📘 # ДЗ 44.1 — Використання React Context у багаторівневій архітектурі компонентів

## 📌 Опис проєкту

Цей проєкт створений у межах домашнього завдання **№44.1** та демонструє використання **React Context API** для передачі глобальних даних між компонентами без використання пропсів.

У проєкті реалізовано:
- створення та налаштування власного контексту;
- використання `Context Provider`;
- доступ до даних контексту на 2–3 рівнях вкладеності компонентів за допомогою хука `useContext`;
- багаторівневу архітектуру компонентів.


## 🛠️ Використані технології

- React
- React Context API
- Vite
- JavaScript (ES6+)
- HTML / CSS
